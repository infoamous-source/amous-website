## 미완료 작업 (사용자가 직접 수행 필요)

### 1. Supabase Storage "images" 버킷 생성 (필수)
이미지 업로드 기능이 동작하려면 반드시 필요합니다.

**방법:**
1. [Supabase 대시보드](https://supabase.com/dashboard) 접속
2. 프로젝트 선택
3. 좌측 메뉴 **Storage** 클릭
4. **New bucket** 클릭
5. 이름: `images`
6. **Public bucket** 체크 ✅
7. **Create bucket** 클릭

### 2. 환경 변수 확인 (필수)
`.env.local` 파일에 아래 값들이 설정되어 있어야 합니다:
```
NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJxxxxx
ADMIN_PASSWORD=관리자비밀번호
JWT_SECRET=랜덤시크릿키
```

### 3. Supabase 데이터 확인 (권장)
- services 테이블의 `icon` 컬럼에 값이 있는지 확인
  - immigrant → `globe`
  - senior → `heart`
  - career → `rocket`
- services 테이블의 `curriculum` 컬럼이 추가되었는지 확인
- 아이콘이 표시되지 않으면 Supabase 대시보드에서 직접 수정
